Station Admin für laut.fm unterstützt Dich bei der Administration Deiner [laut.fm](https://laut.fm)-Station. Als Desktop-Client verfolgt Station Admin andere Ansätze als laut.fm's Radioadmin und bietet daduch oft höheren Komfort und zusätzliche Funktionalität.

Station Admin ist eine Java-Applikation und läuft dadurch sowohl unter Windows wie auch unter Linux oder MacOS. 

Station Admin bietet u. a. folgende Funktionen
- Komfortables Bearbeiten von Playlists
- Konfiguration der [Station Admin-Shuffle-Scripte](https://github.com/frank-lautfm/stationadmin-shuffle)
- Analyse von gespielten Titeln und Hörerzahlen
- Durchsuchen von Playlists
- Backup von Playlists und Sendeplan
- Upload von Titeln

Weitere Informationen findest Du im [Wiki](https://github.com/frank-lautfm/stationadmin/wiki)

Station Admin ist im Juni 2026 von Sourceforge.net zu Github umgezogen. Ältere Versionen sowie deren Sourcecode findest Du nach wie vor auf [sourceforge.net](https://sourceforge.net/projects/stationadmin/files/)
