Das Wichtigste in Kürze

Synchronisieren

  • Nach dem ersten Start Server/Synchronisieren aufrufen - das lädt die Playlists vom laut.fm-Server in lokale Dateien. Vorher funktioniert so gut wie nichts. Das Synchronisieren kann einige Sekunden dauern.
  • Nach jeder Änderung auf laut.fm sollte erneut „Synchronisieren“ aufgerufen werden.

Übersicht

  • Sendeplan: Doppelklick auf eine Sendung öffnet die entsprechende Playlist
  • Letzte Titel: Es wird zunächst die Liste von der laut.fm-Seite ohne Hörerzahl angezeigt. Erst für Titel die nach dem Programmstart gespielt werden steht die Hörerzahl zum Zeitpunkt des Starts zur Verfügung.
  • Kommentare: Es werden die letzten 3 Kommentare angezeigt. Die Liste wird alle 30 Minuten aktualisiert.
Statusleiste
  • Ein Doppelklick auf die aktuelle Sendung öffnet die entsprechende Playlist
  • Ein Doppelklick auf den aktuellen Titel springt an die entsprechende Stelle der aktuellen Playlist
Listen und Tabellen
  • In den meisten Listen und Tabellen kann mit Strg-F gesucht werden
  • Die Tabellen können durch klicken auf die Spaltennamen umsortiert werden.

Protokollierung

  • Um Hörerzahlen und Titel in Dateien mitzuprokollieren bitte in Tools / Einstellungen entsprechende Dateien konfigurieren. %day% kann als Platzhalter für das aktuelle Datum verwendet werden.
