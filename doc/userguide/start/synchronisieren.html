Synchronisieren

Wozu dient das Synchronisieren?

Station Admin speichert eine Kopien Deiner Playlists und Deines Sendplans sowie eine Liste der von Dir hochgeladen Titel lokal auf der Festplatte ab und arbeitet mit diesen Kopien. Wenn nun außerhalb von Station Admin - z. B. via Radioadmin - Playlists oder der Sendeplan geändert wurde oder neue Titel hochgeladen wurden sind die Kopien in Station Admin veraltet.

Beim Synchronisieren werden

  • alle Playlists neu geladen
  • der Sendeplan neu geladen
  • eine Liste neuer eigener Uploads heruntergeladen

Beim Synchroniseren werden übrigens keine Daten von Station Admin an laut.fm übertragen. Wenn Du in Station Admin eine Playlist oder den Sendeplan änderst werden diese Änderungen sofort an den laut.fm-Server übertragen.

Wie kann ich eine Synchronisierung durchführen?

Du kannst jederzeit eine Synchronisierung durchführen in dem Du im Menü Server den Menüpunkt Synchronisieren auswählst. Das muß nach dem ersten Start von Station Admin auf jeden Fall einmal gemacht werden.

Nach dem Programmstart überprüft Station Admin automatisch alle 30 Minuten ob sich die Playlists auf dem laut.fm-Server geändert haben. Wenn eine Änderung erkannt wurde fragt Station Admin nach ob eine Synchronisierung gewünscht ist und führt diese beim Bestätigen sofort durch.
Die automatische Erkennung hat allerdings ihre Grenzen: Wenn sich die Länge mindestens einer Playlist geändert hat wird eine Änderung erkannt. Wurden nur Titel vertauscht, die Länge aber unverändert gelassen, wird Station Admin das nicht erkennen.

Wozu dient "Eigene Titel neu laden"?

Im Menü Server gibt es neben Synchronisieren auch noch Eigenen Titel neu laden. Damit wird die Liste der eigenen Titel komplett neue geladen und nicht - wie beim Synchronisieren - nur nach neuen Titeln geschaut. Das kann notwendig sein wenn Titel via Radioadmin oder durch laut.fm geändert oder gelöscht wurden.

Die eigenen Titeln komplett neu zu laden ist eine relativ teure Aktion - bei manchen Stations sind das mehrere tausend Titel -, deshalb wird das nicht automatisch im Rahmen der Synchronisierung gemacht.
