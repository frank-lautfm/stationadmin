Installation

Bevor es los geht

Station Admin läuft auf allen gängigen Betriebssystemen. Entwickelt und getestet wird Station Admin auf Windows XP und Vista. Es gibt aber auch DJs die Station Admin erfolgreich auf einem Mac einsetzen - hierzu ist allerdings mindestens MacOS 10.7 erforderlich.
Wichtig ist dass Du eine Java Runtime Environment der Version 7 oder höher installiert hast. Die gibt es hier.

Download und Installation

Falls Du das nicht schon getan hast, lade die aktuelle Version von Station Admin bitte von hier herunter.
Station Admin wird als zip-Datei ausgelieft. Diese Datei kannst Du in ein beliebiges Verzeichnis entpacken. Ein Installationsprogramm gibt es nicht - das wird auch nicht benötigt.

Station Admin starten

Windows-Benutzer können Station Admin durch Doppelklick auf StationAdmin.exe starten. Auf anderen Betriebssystem - z. B. unter MacOS - funktioniert üblicherweise ein Klick / Doppelklick auf StationAdmin.jar.

Sollten die Varianten oben bei Dir nicht funktionieren hilft notfalls noch der Aufruf aus einer Kommandozeile: java -jar StationAdmin.jar
