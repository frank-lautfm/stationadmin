Tagesstatistik

Überblick

laut.fm stellt einige statistische Werte für den Vortag zur Verfügung: Gesamthördauer, Einschaltvorgänge und durchschnittliche Hördauer. Station Admin protokolliert diese Werte mit.

Um die Statistiken vorheriger Tage aufzurufen wähle im Menü Analyse/Tagestatistik.

Filter

Oben kannst Du den Zeitraum auswõhlen f³r den die Eintrõge angezeigt werden sollen. Änderungen werden erst nach einem Klick auf den Filtern-Button rechts wirksam.

Anzeige

In der Tabelle werden - insofern verfügbar - für jeden Tag der innerhalb des gewählten Zeitraums liegt die folgenden Werte angezeigt:

  • Gesamthördauer in Stunden
  • Zahl der Einschaltvorgänge
  • Durchschnittliche Hördauer in Minuten
Die Werte für einen Tag stehen natürlich nur zur Verfügung wenn Station Admin am Folgetag irgendwann lief und die Möglichkeit hatte die Werte zu protokollieren. Stehen für einen Tag keine protokollierten Werte zur Verfügung versucht Station Admin die Gesamthördauer auf Basis der Hörerzahlen abzuschätzen - dies setzt allerdings die Erlaubnis vorraus diese Zahlen von einem externen Server abzurufen. Die hochgerechnete Hördauer ist ungenau und wird in kursiver Schrift dargestellt.

In der Statusleiste unter der Tabelle werden Durschschnittswerte für Gesamthördauer, Einschaltvorgänge und durschnittlicher Hördauer angezeigt.
