Einstellungen auf dem Server speichern

Station Admin erfasst zahlreiche Konfigurationsdaten die über das hinaus gehen was der Radioadmin speichert. Die beinhaltet

  • Grundeinstellungen für Playlists
  • Regeln zum Befüllen von Playlists
  • diverse Playlisteigenschaften wie Playlist-Tags oder Notizen
  • Gruppenzordnung von Tags
  • Filterregeln für Titel (früher: dynamische Tags)

Wenn Du auf mehreren Rechnern mit Station Admin arbeitest oder wenn Ihr mit mehreren Personen eine Station betreut ist es natürlich sinnvoll wenn alle Benutzer und alle Rechner die gleichen Einstellungen für die Station sehen. Station Admin kann diese Einstellungen deshalb auf dem Server ablegen. Bei Neuinstallationen ist das automatisch aktiviert, wenn Du ein Upgrade von Version 4.x machst musst Du das selbst aktivieren.

Um das Speichern der Konfigurationsdaten auf dem Server zu aktivieren oder deaktiveren öffne im Menü Extras/Einstellungen. Auf dem Reiter Allgemein findest Du den Bereich Konfiguratiosndaten mit einer entsprechenden Checkbox. Wenn Du die Einstellugen aktivierst schaut Station Admin nach ob auf dem Server schon Einstellungen gespeichert sind. Ist dies der Fall wirst Du gefragt ob Du diese jetzt laden möchtest. Lehnst Du das ab oder wurden keine Einstellungen auf dem Server gefunden werden Deine aktuellen Einstellungen auf dem Server abgelegt.
