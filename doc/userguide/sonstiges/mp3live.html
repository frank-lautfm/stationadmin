MP3-Dateien live senden

Wenn Deine Station für das Live-Senden freigeschaltet ist kannst Du über Station Admin vorproduzierte Sendungen oder Mixe die im MP3-Format vorliegen 'live' senden. Das kann entweder manuell gestartet werden oder als Aufgabe eingerichtet werden und zu einem vorgegebenen Zeitpunkt gestartet werden.

Übertragung manuell starten

Um eine Übertragung manuell zu starten klicke im Server-Menü auf MP3-Datei 'live' senden.

Folgende Informationen müssen oder können angegeben werden:

  • MP3-Datei: Wähle die Datei aus die Du senden möchtest. Wichtig: Achte auf das Format: 128kbps, 44 Mhz
  • Titelinformationen: Wähle eine Datei aus die Titelinformationen für die MP3-Datei enthält. Dies ist optional, die Übertragung kann auch ohne Titelinformationen gestartet werden.
  • Aktiviere voherigen Titel zu Ende spielen lassen wenn die Übertragung erst gestartet werden soll nach dem der aktuelle Titel zu Ende ist.

Titelinformationen können in einer Textdatei bereitgestellt werden die eine Zeile pro Titel enthält. Jeder Eintrag sollte im Format Startzeit Artist - Titel angegeben werden. Ein Beispiel: 

0:00 Secret Garden - Lament For A Frozen Flower
5:02 Oystein Sevag & Lakki Patey - Painful Love
8:37 James Newton Howard - I'm Listening

Für Dateien die länger als 20 Minuten gehen sollte währden der Übertragund der Werbetrigger ausgelöst werden. Das geschieht in dem ein Titel "START_AD_BREAK - START_AD_BREAK" eingetragen wird. Es empfiehlt sich diesen nur für eine Sekunde einzufügen - also den nächsten regulären Titel ganz normal eine Sekunde später einzutragen.

Alternativ können die Werbetrigger auch automatisch eingefügt werden. Aktiviere dazu die Checkbox Werbetrigger einfügen nach... und gebe zwei Positionen an die mindestens 20, aber höchstens 40 Minuten auseinander liegen. Ist Deine Datei länger als eine Stunde wiederholen sich die Werbetrigger, also z. B. nach 0:20, 0:50, 1:20, 1:50 etc

Klicke auf Starten um die Übertragung zu beginnen. Eine laufende Übertragung kann jederzeit durch klicken auf Stoppen abgebrochen werden. Während eine Übertragung läuft werden im mittleren Bereich einige Informationen angezeigt:

  • Status: Mögliche Werte sind 'online', 'offline' und 'warten' (wenn der vorherige Titel noch läuft)
  • Zeit: Zeit seit dem Start der Übertragung. Dies bezieht sich auf den Zeitpunkt ab dem die Daten zu Icecast übertragen wurde - Icecast beginnt erst einige Sekunden später mit der wirklichen Übertragung an Clients.
  • Titel: Insofern eine Datei mit Titelinformationen vorliegt wird hier der aktuelle Titel angezeigt.

Einige Hinweise:
  • Die Dialogbox für die Liveübertragung kann jederzeit geschlossen und später wieder geöffnet werden.
  • Station Admin überträgt bis zu 15 Sekunden im Vorraus an Icecast. D. h. wenn die Übertragung gestoppt wird wird Icecast trotzdem noch mindestens 15 Sekunden das Liveprogramm weiter senden.
  • Da es einen kurzen Moment dauert bis Icecast nach dem Verbindungsaufbau mit den Senden beginnt wird die Übertragung 3 Sekunden vor dem Ende des aktuellen Titels gestartet wenn dieser zu Ende gespielt werden soll. Das hat sich in Tests als guter Wert erwiesen - es kann aber sein dass dem aktuellen Titel doch mal die letzten ein oder zwei Sekunden fehlen.

Übertragung automatisch starten

Um eine Übertragung zu einem vorgegeben Zeitpunkt zu starten lege eine Aufgabe an - das ist hier genauer beschrieben. Die Konfigurationsmöglichkeiten entsprechen denen beim manuellen Start der Übertragung.
